# Get-CUCommands

## SYNOPSIS
Returns the list of commands available in the PSclassUtils module

## SYNTAX

```
Get-CUCommands [<CommonParameters>]
```

## DESCRIPTION
All public commands will be returned.

## EXAMPLES

### EXAMPLE 1
```
Get-CUCommands
```
